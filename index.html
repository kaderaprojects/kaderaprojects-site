Welcome to TheKader Blog

This is the first post. I’ll use this space to document projects, learning paths, home lab work, and anything worth sharing as I build.

Cybersecurity has always been a core part of my work and my curiosity. I enjoy breaking down complex vulnerabilities, testing tools in my home lab, and learning how attackers think so I can build stronger defenses. My goal with this blog is to share what I learn, explain concepts in plain language, and help others strengthen their own security skills.

Over time, I’ll be posting walkthroughs, security notes, and my thoughts on real-world risks, cloud security, and building a strong cyber mindset.
